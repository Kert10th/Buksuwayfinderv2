# Campus Map Asset

## File Required
Please add your campus map image file as:
- **Filename:** `buksu-campus-map-main.png`
- **Location:** `src/assets/buksu-campus-map-main.png`

## Current Setup
The wayfinder is configured to use `buksu-campus-map-main.png` as the main campus map.

Once you add the image file, it will automatically be displayed in the WayfinderInterface component.

## Image Specifications
- Recommended format: PNG (for transparency support)
- Recommended resolution: High resolution for zoom functionality
- The map will be displayed with zoom and pan capabilities
